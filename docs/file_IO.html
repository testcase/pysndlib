File IO#

in pysndlib.clm

file2ndarray(filename, channel=None, beg=None, dur=None)#

return an ndarray with samples from file and the sample rate of the data

Parameters:

  • filename (str) – filename

  • channel (Optional[int]) – if None, will read all channels, otherwise just channel specified

  • beg (Optional[int]) – beginning positions to read in samples

  • dur (Optional[int]) – duration in samples to read

Returns:

tuple of np.ndarray and sample rate

Return type:

tuple

ndarray2file(filename, arr, length=None, sr=None, sample_type=Sample.BFLOAT, header_type=Header.AIFC, comment=None)#

write an ndarray of samples to file

Parameters:

  • filename (str) – name of file

  • arr (npt.NDArray[np.float64]) – np.ndarray of samples

  • length – length of samples to write. if None write all

  • sr – sample rate of file to write

  • sample_type (Optional[sndlib.sample]) – type of sample type to use. defaults to clm.sample_type

  • header_type (Optional[sndlib.header]) – header of sample type to use. defaults to clm.header_type

Returns:

length in samples of file

Return type:

int
